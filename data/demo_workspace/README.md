# Demo Workspace Notes

- This directory keeps only placeholder files and lightweight notes needed for the demo workspace layout.
- The file `should_not_run.txt` is a local placeholder created in the workspace root for agent-behavior checks.
- Files generated by smoke tests or MCP runs stay local and are not tracked in Git.
- Ad-hoc manual test files under `manual_test/` also stay local and are not tracked in Git.
- GUI-created `gui_*` task directories also stay local and are not tracked in Git.
